# be_ms_car_rc_platform

Service working with cars entity. For storing and processing data of the cars.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger

Swagger documentation can be checked in `http://localhost:3000/api`
